# Walkthrough - Phase 64

## What Changed
This phase focused on making the AI Agent feel safer and more deliberate before a user enters the messaging flow.

## Selection Guidance
The selection banner now reads more like workflow guidance instead of a send prompt.
That matches the intent that users may keep managing selected records before they ever send a message.

## Image Preview
Template images are now easier to inspect in both the AI Agent and Send Message flows through a dedicated preview/lightbox interaction.

## Template-Aware Send Flow
If a user explicitly asks to send with a template, the AI Agent now checks whether a current template is actually available in context.
If not, it guides the user to open or manage one first.
