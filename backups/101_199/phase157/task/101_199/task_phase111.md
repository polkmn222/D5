# Phase 111 Task

## Context

The AI Recommend mode label `Default` is too vague for end users, and the recommendation table should not feel capped at five rows when more sendable deals exist.

## Goals

- Rename the user-facing `Default` recommendation mode to a clearer label.
- Keep compatibility with existing AI Agent commands while presenting the clearer label in the UI and help text.
- Expand the shared sendable recommendation list size and render it in a scrollable table when more rows are available.
