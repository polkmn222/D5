# Phase 110 Task

## Context

The AI Agent selection bar currently only exposes `Send Message`, while users also need separate manual actions for editing and deleting a selected record. The quick guide is also missing a direct Send Message shortcut, and the meaning of the AI Recommend `Default` mode should be clearer.

## Goals

- Add separate `Edit` and `Delete` actions to the AI Agent selection bar.
- Add a `Send Message` quick guide button.
- Clarify that AI Recommend `Default` means the most recently created sendable deals.
- Add unit coverage for the updated AI Agent UI and recommendation copy.
