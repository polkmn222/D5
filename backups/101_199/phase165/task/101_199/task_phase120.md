# Phase 120 Task

## Context

The AI Agent selection experience still needs refinement: `Open` and `Edit` should stay inside the AI Agent workflow, delete confirmation should use visible names instead of raw IDs, the confirmation buttons should feel more actionable, and the selection action box should appear at the bottom of the active result table.

## Goals

- Keep `Open` and `Edit` inside the AI Agent context.
- Replace ID-based selection and delete copy with record names.
- Move the selection action box to the bottom of the active table.
- Make `[yes]` / `[cancel]` confirmation buttons larger and clearer.
